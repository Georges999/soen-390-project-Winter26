# UI Mockups and Design

This folder contains all UI/UX design files, mockups, and wireframes.

## Contents

### Wireframes
Low-fidelity sketches of screen layouts and user flows

### Mockups
High-fidelity designs showing final UI appearance

### Design System
- Color palette (Concordia brand colors)
- Typography
- Component library
- Icons and imagery

## Concordia Branding

### Colors
- **Primary:** #912338 (Concordia Red)
- **Secondary:** #FFD700 (Gold accent)
- **Background:** #FFFFFF (White)
- **Text:** #333333 (Dark gray)
- **Accent:** #4A90E2 (Blue for interactive elements)

### Typography
- **Headings:** Bold, 24-32px
- **Body:** Regular, 16px
- **Small text:** 12-14px

## Screens to Design

1. **Splash Screen** - App loading screen
2. **Home/Map Screen** - Main campus map view
3. **Directions Screen** - Route display and navigation
4. **Schedule Screen** - Class schedule overview
5. **Indoor Navigation Screen** - Floor plan and pathfinding
6. **Points of Interest** - Search and filter locations
7. **Settings Screen** - App preferences and accessibility

## Design Tools
- [Figma](https://figma.com) - Collaborative design tool (recommended)
- [Adobe XD](https://www.adobe.com/products/xd.html)
- [Sketch](https://www.sketch.com/)

## Export Guidelines
- Export mockups as PNG (high resolution)
- Include source files (.fig, .xd, .sketch)
- Name files clearly: `screen-name-state.png`
  - Example: `home-screen-default.png`, `home-screen-loading.png`
